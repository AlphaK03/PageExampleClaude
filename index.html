¿Qué son los algoritmos de ordenamiento?

Son métodos para reorganizar elementos de una lista en un orden determinado (ascendente, descendente, etc.). Son fundamentales en ciencias de la computación y aparecen en casi cualquier aplicación real.

Bubble Sort

O(n²)

Compara pares adyacentes y los intercambia si están en el orden incorrecto. Simple pero ineficiente.

Tiempo: O(n²) Espacio: O(1) 
function bubbleSort(arr) {
  const n = arr.length;
  for (let i = 0; i < n-1; i++) {
    for (let j = 0; j < n-i-1; j++) {
      if (arr[j] > arr[j+1]) {
        [arr[j], arr[j+1]] =
          [arr[j+1], arr[j]];
      }
    }
  }
  return arr;
}

Selection Sort

O(n²)

Encuentra el elemento mínimo del subarreglo no ordenado y lo mueve al inicio.

Tiempo: O(n²) Espacio: O(1) 
function selectionSort(arr) {
  const n = arr.length;
  for (let i = 0; i < n; i++) {
    let min = i;
    for (let j = i+1; j < n; j++) {
      if (arr[j] < arr[min])
        min = j;
    }
    if (min !== i)
      [arr[i], arr[min]] =
        [arr[min], arr[i]];
  }
  return arr;
}

Insertion Sort

O(n²) / O(n)*

Inserta cada elemento en su posición correcta dentro del subarreglo ya ordenado. Eficiente para listas casi ordenadas.

Tiempo: O(n²) Espacio: O(1) 
function insertionSort(arr) {
  for (let i = 1; i < arr.length; i++) {
    const key = arr[i];
    let j = i - 1;
    while (j >= 0 && arr[j] > key) {
      arr[j + 1] = arr[j];
      j--;
    }
    arr[j + 1] = key;
  }
  return arr;
}

Merge Sort

O(n log n)

Divide el arreglo en mitades, ordena cada mitad recursivamente y luego las combina. Estable y predecible.

Tiempo: O(n log n) Espacio: O(n) 
function mergeSort(arr) {
  if (arr.length <= 1) return arr;
  const mid = Math.floor(arr.length/2);
  const L = mergeSort(arr.slice(0,mid));
  const R = mergeSort(arr.slice(mid));
  return merge(L, R);
}

function merge(L, R) {
  const res = [];
  let i = 0, j = 0;
  while (i < L.length && j < R.length)
    res.push(L[i]<=R[j] ? L[i++] : R[j++]);
  return [...res, ...L.slice(i), ...R.slice(j)];
}

Quick Sort

O(n log n)*

Elige un pivote y particiona el arreglo. Muy rápido en la práctica, aunque O(n²) en el peor caso.

Tiempo: O(n log n) Espacio: O(log n) 
function quickSort(arr, lo=0, hi=arr.length-1) {
  if (lo >= hi) return arr;
  const p = partition(arr, lo, hi);
  quickSort(arr, lo, p-1);
  quickSort(arr, p+1, hi);
  return arr;
}

function partition(arr, lo, hi) {
  const pivot = arr[hi];
  let i = lo;
  for (let j=lo; j<hi; j++) {
    if (arr[j] <= pivot)
      [arr[i++], arr[j]] = [arr[j], arr[i]];
  }
  [arr[i], arr[hi]] = [arr[hi], arr[i]];
  return i;
}
